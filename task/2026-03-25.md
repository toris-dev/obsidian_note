1. **프로모션 탐색**
2. **프로모션 지원**
3. **병원 검토**
4. **매칭 생성 (Match 생성)**
5. **모델 수락**
6. **시술 일정 확정 (병원)**
7. **계약 확인 + 보증금 납부 (모델)**
8. **시술 진행**
9. **미션 수행 (리뷰)**
10. **미션 승인 → 보증금 환불**

# 🚀 2. 병원 → 모델 플로우

## 💡 핵심: “컨택 = 프로모션 제안”

1. **병원이 모델 탐색**
2. **프로모션 선택**
3. **모델에게 제안 (컨택 X, 제안 O)**
4. **모델 수락**
5. **매칭 생성 (또는 수락 시 생성)**
6. 이후는 완전히 동일 👇

- 시술 일정 확정
- 계약 + 보증금
- 시술
- 미션
- 환불
